Перед тем как перейти к следующим примерам обратим внимание на то, что вычисление интеграла с помощью теоремы о дифференцируемости обязательно содержит «обратный ход», т.е. восстановление функции по ее производной. При этом каждой дифференцируемой функции \(\displaystyle f\left( x \right)\) однозначно соответствует ее производная \(\displaystyle {f}'\left( x \right)\). Но обратное утверждение неверно, так как \[\forall C\in \mathbb{R}\,\,\,{{\left( f\left( x \right)+C \right)}^{\prime }}={f}'\left( x \right).\]

Поэтому, чтобы получить \(\displaystyle f\left( x \right) \text{ из } {f}'\left( x \right)\) надо знать значение \(\displaystyle f\left( x \right)\) хотя бы в одной точке. Если известно значение в точке \(\displaystyle {{x}_{0}},\) то функцию можно найти двумя способами, используя:

     – определенный интеграл: \[f\left( x \right)=\int\limits_{{{x}_{0}}}^{x}{{f}'\left( x \right)dx};\]

     – неопределенный интеграл: \[f\left( x \right)+C=\int{{f}'\left( x \right)dx}, \text{ где } C=-f\left( {{x}_{0}} \right).\]

Эти 2 способа имеют одинаковую трудоемкость и по сути являются лишь разными способами записи одних и тех же действий (это становится понятно, если записать для определенного интеграла формулу Ньютона-Лейбница).

Доказательство

Рассмотрим сначала более общий интеграл

\begin{eqnarray*} \displaystyle I\left( a,b \right)=\int\limits_{0}^{+\infty }{{{e}^{-bx}}\frac{\sin ax}{x}dx}, \displaystyle b\in \left[ 0,1 \right]. \end{eqnarray*}

Этот интеграл при каждом фиксированном \(\displaystyle a\ne 0\) сходится равномерно по параметру \(\displaystyle b\) на отрезке \(\displaystyle \left[ 0,1 \right]\) по признаку Дирихле:

        1) \(\displaystyle \left| \int\limits_{0}^{B}{\sin ax\,dx} \right|=\left| \left. \frac{\cos ax}{a} \right|_{0}^{B} \right|\le \frac{2}{\left| a \right|}<+\infty \),

        2) \(\displaystyle \frac{{{e}^{-bx}}}{x}\) монотонно убывает по \(\displaystyle x\),

        3) \(\displaystyle \frac{{{e}^{-bx}}}{x}\underset{\left[ 0,1\right]}\rightrightarrows\,0\) при \(\displaystyle x\to \infty \), так как \(\displaystyle \underset{x\to +\infty }{\mathop{\lim }}\,\underset{b\in \left[ 0,1 \right]}{\mathop{\sup }}\,\left| \frac{{{e}^{-bx}}}{x}-0 \right|=\underset{x\to +\infty }{\mathop{\lim }}\,\frac{1}{x}=0\).

Из равномерной сходимости интеграла \(\displaystyle I\left( a,b \right)\) и непрерывности функции \begin{eqnarray*} \displaystyle \left\{ \begin{matrix} {{e}^{-bx}}\frac{\sin ax}{x},\,\,x>0 \\ a,\,\,x=0 \\ \end{matrix} \right. \end{eqnarray*} на множестве \[\displaystyle \left\{ \left. \left( x,b \right)\, \right|\,\,x\in \left[ 0,+\infty \right),\,\,b\in \left[ 0,1 \right] \right\}\]

можем утверждать (в силу теоремы о непрерывности несобственного интеграла), что функция \(\displaystyle I\left( a,b \right)\) непрерывна по \(\displaystyle b\) на \(\displaystyle \left[ 0,1 \right]\), значит: \begin{eqnarray*} \displaystyle \int\limits_{0}^{+\infty }{\frac{\sin ax}{x}dx}=\underset{b\to +0}{\mathop{\lim }}\,\int\limits_{0}^{+\infty }{{{e}^{-bx}}\frac{\sin ax}{x}dx}=\underset{b\to +0}{\mathop{\lim }}\, \operatorname{arctg}\frac{a}{b}=\frac{\pi }{2}\,\,\operatorname{sign}a. \end{eqnarray*}
