Интегралы, зависящие от параметра Интегралы Эйлера

Интегралы Эйлера


Гамма-функция \(\displaystyle \Gamma\left( p \right)=\int\limits_{0}^{+\infty }{{{e}^{-x}}{{x}^{p-1}}dx}\)


Сходимость

Из теоремы сравнения следует, что при \(\displaystyle p\le 0\) интеграл расходится. Представим функцию \(\displaystyle \Gamma\left( p \right)\) в виде

\(\displaystyle \Gamma\left( p \right)=\int\limits_{0}^{1}{{{e}^{-x}}{{x}^{p-1}}dx}+\int\limits_{1}^{+\infty }{{{e}^{-x}}{{x}^{p-1}}dx}={{I}_{1}}\left( p \right)+{{I}_{2}}\left( p \right)\).

1. Так как \(\displaystyle {{e}^{-x}}{{x}^{p-1}}\le {{x}^{p-1}}\) при \(\displaystyle x>0\), а \(\displaystyle \int\limits_{0}^{1}{{{x}^{p-1}}dx}\) сходится при \(\displaystyle p>0\), то и интеграл \(\displaystyle {{I}_{1}}\left( p \right)\) сходится при \(\displaystyle p>0\) (по признаку Вейерштрасса).

2. Так как \(\displaystyle \underset{x\to +\infty }{\mathop{\lim }}\,{{e}^{-x}}{{x}^{r}}=0\) при любом \(\displaystyle r\), то из сходимости интеграла \(\displaystyle \int\limits_{1}^{+\infty }{{{x}^{-2}}dx}\) следует сходимость интеграла \(\displaystyle {{I}_{2}}\left( p \right)\) при любом \(\displaystyle p\). (Из определения предела

\(\displaystyle \forall r>0\,\,\exists {{x}_{0}}\,\forall x>{{x}_{0}}\ \ {{e}^{-x}}<\frac{1}{{{x}^{r}}}\),

следовательно,

\(\displaystyle \forall p>0\,\,\exists {{x}_{0}}\ \forall x>{{x}_{0}}\ \ {{e}^{-x}}<\frac{1}{{{x}^{p+1}}}\ \ \Leftrightarrow \ \ {{e}^{-x}}{{x}^{p-1}}<\frac{1}{{{x}^{1-p}}}\frac{1}{{{x}^{p+1}}}=\frac{1}{{{x}^{2}}}\)).


Непрерывность

Для доказательства непрерывности \(\displaystyle \Gamma\left( p \right)\) на полупрямой \(\displaystyle p>0\), достаточно установить равномерную сходимость интеграла относительно параметра \(\displaystyle p\) при \(\displaystyle 0\,<\,{{p}_{0}}\,\le \,p\,\le \,{{p}_{1}}\) для любых фиксированных значений \(\displaystyle {{p}_{0}}\) и \(\displaystyle {{p}_{1}}\), удовлетворяющих условию \(\displaystyle 0<{{p}_{0}}\le {{p}_{1}}\).

Так как

     1) при \(\displaystyle 0<x<1\) степенная функция \(\displaystyle {{x}^{p}}\) убывает, то \[\displaystyle {{x}^{p}}\le {{x}^{{{p}_{0}}}}<{{x}^{{{p}_{0}}}}+{{x}^{{{p}_{1}}}},\]

     2) при \(\displaystyle x≥1\) степенная функция \(\displaystyle {{x}^{p}}\) возрастает, то \[\displaystyle {{x}^{p}}\le {{x}^{{{p}_{1}}}}<{{x}^{{{p}_{0}}}}+{{x}^{{{p}_{1}}}},\]

поэтому

\(\displaystyle \forall x>0 \quad {{x}^{{{p}}}}\le{{x}^{{{p}_{0}}}}+{{x}^{{{p}_{1}}}}\).

умножив последнее неравенство на \(\frac{{{e}^{-x}}}{x}\), получим

\(\displaystyle \forall x>0 \quad \frac{{{e}^{-x}}}{x}{{x}^{p}}\le \frac{{{e}^{-x}}}{x}\left[ {{x}^{{{p}_{0}}}}+{{x}^{{{p}_{1}}}} \right]={{e}^{-x}}\left[ {{x}^{{{p}_{0}}-1}}+{{x}^{{{p}_{1}}-1}} \right]\).

Равномерная сходимость интеграла \(\displaystyle \Gamma\left( p \right)\) следует в силу признака Вейерштрасса из сходимости интеграла

\(\displaystyle \int\limits_{0}^{+\infty }{{{e}^{-x}}\left[ {{x}^{{{p}_{0}}-1}}+{{x}^{{{p}_{1}}-1}} \right]dx}=\Gamma\left( {{p}_{0}} \right)+\Gamma\left( {{p}_{1}} \right)\).


Формула приведения

Применяя формулу интегрирования по частям для функции \(\displaystyle \Gamma\left( p+1 \right)\)при \(\displaystyle p>0\) получим

\(\displaystyle \Gamma\left( p+1 \right)=\int\limits_{0}^{+\infty }{{{e}^{-x}}{{x}^{p}}dx}=\left. \left[ -{{e}^{-x}}{{x}^{p}} \right] \right|_{0}^{+\infty }+p\int\limits_{0}^{+\infty }{{{e}^{-x}}{{x}^{p-1}}dx}=p\Gamma\left( p \right)\).


Бета-функция \(\displaystyle B\left( p,q \right)=\int\limits_{0}^{1}{{{x}^{p-1}}{{\left( 1-x \right)}^{q-1}}dx}\)


Сходимость

Из теоремы сравнения следует, что при \(\displaystyle p\le 0\), \(\displaystyle q\le 0\) интеграл расходится.

1. При \(\displaystyle p\ge 1\) и \(\displaystyle q\ge 1\) подынтегральная функция в соотношении непрерывна, поэтому интеграл в правой части является собственным. Следовательно, функция \(\displaystyle B\left( p,q \right)\) определена для всех \(\displaystyle p\ge 1\), \(\displaystyle q\ge 1\).

2. Пусть выполняются одно или оба из следующих неравенств

\(\displaystyle 0<p<1\), \(\displaystyle 0<q<1\).

Тогда одна или обе точки \(\displaystyle x=0\) и \(\displaystyle x=1\) являются особыми точками подынтегральной функции. Представим функцию \(\displaystyle B\left( p,q \right)\) в виде

\(\displaystyle B\left( p,q \right)=\int\limits_{0}^{1/2}{{{x}^{p-1}}{{\left( 1-x \right)}^{q-1}}dx}+\int\limits_{1/2}^{1}{{{x}^{p-1}}{{\left( 1-x \right)}^{q-1}}dx}={{I}_{1}}\left( p,q \right)+{{I}_{2}}\left( p,q \right)\).

Для интеграла \(\displaystyle {{I}_{1}}\left( p,q \right)\) особой точкой будет точка \(\displaystyle x=0\). На сегменте \(\displaystyle \left[ 0,1/2 \right]\) функция \(\displaystyle {{\left( 1-x \right)}^{q-1}}\) непрерывна и поэтому ограничена некоторой константой \(\displaystyle C\), тогда функция \(\displaystyle C{{x}^{p-1}}\) будет мажорантой для подынтегральной функции интеграла \(\displaystyle {{I}_{1}}\left( p,q \right)\). Отсюда следует, что интеграл \(\displaystyle {{I}_{1}}\left( p,q \right)\) сходится при \(\displaystyle 0<p<1\) и любом \(\displaystyle q\). Аналогично доказывается, что интеграл \(\displaystyle {{I}_{2}}\left( p,q \right)\) сходится при \(\displaystyle 0<q<1\) и любом \(\displaystyle p\).

Объединяя случаи 1 и 2, получаем, что функция \(\displaystyle B\left( p,q \right)\) определена для \(\displaystyle p>0,\ \ q>0\).


Непрерывность

Для доказательства непрерывности \(\displaystyle B\left( p,q \right)\) в квадранте \(\displaystyle p>0,\ \ q>0\) достаточно доказать равномерную сходимость интеграла относительно параметров \(\displaystyle p\) и \(\displaystyle q\) при \(\displaystyle p\ge {{p}_{0}}>0\) и \(\displaystyle q\ge {{q}_{0}}>0\) для любых фиксированных значений \(\displaystyle {{p}_{0}}\) и \(\displaystyle {{q}_{0}}\). При \(\displaystyle 0<x<1\) справедливо неравенство

\(\displaystyle {{x}^{p-1}}{{\left( 1-x \right)}^{q-1}}\le {{x}^{{{p}_{0}}-1}}{{\left( 1-x \right)}^{{{q}_{0}}-1}}\).

Отсюда из сходимости интеграла

\(\displaystyle \int\limits_{0}^{1}{{{x}^{{{p}_{0}}-1}}{{\left( 1-x \right)}^{{{q}_{0}}-1}}dx}=B\left( {{p}_{0}},{{q}_{0}} \right)\)

вытекает в силу признака Вейерштрасса равномерная сходимость интеграла. Таким образом, непрерывность функции \(\displaystyle B\left( p,q \right)\) в квадранте \(\displaystyle p>0,\ \ q>0\) доказана.

Формула приведения

Считая, что \(\displaystyle p>0\), \(\displaystyle q>0\), интегрируем по частям функцию \(\displaystyle B\left( p,q+1 \right)\)

\(\displaystyle \begin{matrix} B\left( p,q+1 \right)=\int\limits_{0}^{1}{{{x}^{p-1}}{{\left( 1-x \right)}^{q}}dx}= \\ =\left. \left[ \frac{{{x}^{p}}}{p}{{\left( 1-x \right)}^{q}} \right] \right|_{0}^{1}+\frac{q}{p}\int\limits_{0}^{1}{{{x}^{p}}{{\left( 1-x \right)}^{q-1}}dx}= \\ =\frac{q}{p}\int\limits_{0}^{1}{\left\{ {{x}^{p-1}}{{\left( 1-x \right)}^{q-1}}-{{x}^{p-1}}{{\left( 1-x \right)}^{q}} \right\}dx}= \\ =\frac{q}{p}B\left( p,q \right)-\frac{q}{p}B\left( p,q+1 \right). \\ \end{matrix}\)

Следовательно,

\(\displaystyle B\left( p,q+1 \right)=\,\frac{q}{p+q}B\left( p,q \right)\).

Учитывая симметрию функции \(\displaystyle B\left( p,q \right)\), получаем вторую формулу приведения

\(\displaystyle B\left( p+1,q \right)=\,\frac{p}{p+q}B\left( p,q \right)\).
