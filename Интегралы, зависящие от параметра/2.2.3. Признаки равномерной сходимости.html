Признаки равномерной сходимости

1. Признак Вейерштрасса.

Пусть функция \(\displaystyle f(x,\gamma )\) определена на множестве \(\displaystyle \left[ a,\mathbf{B} \right)\times\) \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось)., \(\displaystyle \forall \gamma \in \Gamma \) интегрируема по \(\displaystyle x\) на любом сегменте \(\displaystyle \left[ a,\eta \right]\,\subset \,\left[ a,\mathbf{B} \right)\) и равномерно по \(\displaystyle \gamma \) ограничена на \(\displaystyle \left[ a,\mathbf{B} \right)\) (т.е. \(\displaystyle \forall x\,\in \left[ a,\mathbf{B} \right)\,\,\,\underset{\gamma \in \Gamma }{\mathop{\sup }}\,\left| f(x,\gamma ) \right|\,\le \,g(x)\)). Тогда из сходимости интеграла \(\displaystyle \int\limits_{a}^{\mathbf{B}}{g(x)dx}\) следует равномерная по \(\displaystyle \gamma \) на множестве \( \Gamma \) сходимость интеграла \(\displaystyle I(\gamma )\,=\int\limits_{a}^{\mathbf{B}}{f(x,\gamma )dx}\).

Подробнее

Требуемое утверждение получается из неравенства

\(\displaystyle \left| \int\limits_{{{A}'}}^{{{A}''}}{f(x,\gamma )dx} \right|\le \int\limits_{{{A}'}}^{{{A}''}}{\left| f(x,\gamma ) \right|dx}\le \int\limits_{{{A}'}}^{{{A}''}}{g(x)dx}\).

3. Признак Абеля

Пусть функции \(\displaystyle f(x,\gamma )\) и \(\displaystyle g(x,\gamma )\) определены на множестве \(\displaystyle \left[ a,\mathbf{B} \right)\times\) \( \Gamma \)Г - произвольный промежуток (интервал, полуинтервал, отрезок, луч, числовая ось)..

Несобственный интеграл \(\displaystyle \int\limits_{a}^{\mathbf{B}}{f\left( x,\gamma \right)g\left( x,\gamma \right)dx}\) сходится равномерно на \( \Gamma \), если:

1) \(\displaystyle \exists C\in \mathbb{R}\,\)\(\displaystyle \underset{\gamma \in \Gamma }{\mathop{\sup }}\,\underset{a\le \eta <\mathbf{B}}{\mathop{\sup }}\,\left| \int\limits_{a}^{\eta }{f\left( x,\gamma \right)dx} \right|\le C\);

1) \(\displaystyle \int\limits_{a}^{\mathbf{B}}{f\left( x,\gamma \right)dx}\)

сходится равномерно на \( \Gamma \);

2) \(\displaystyle \forall \gamma \in \Gamma \) \(\displaystyle g\left( x,\gamma \right)\) монотонна по \(\displaystyle x\);

3) \(\displaystyle g\left( x,\gamma \right)\underset{\Gamma }\rightrightarrows\,0\) при \(\displaystyle x\to \mathbf{B}\), т.е. \(\displaystyle \underset{x\to \mathbf{B}}{\mathop{\lim }}\,\underset{\gamma \in \Gamma }{\mathop{\sup }}\,\left| g\left( x,\gamma \right) \right|=0\).

3) \(\displaystyle g\left( x,\gamma \right)\) равномерно ограничена на \( \Gamma \), т.е. \(\displaystyle \exists C\in \mathbb{R}\,\)\(\displaystyle \underset{a\le x<\mathbf{B}}{\mathop{\sup }}\,\underset{\gamma \in \Gamma }{\mathop{\sup }}\,\left| g\left( x,\gamma \right) \right|\le C\).

Замечание 1. Признаки Вейерштрасса, Дирихле и Абеля являются достаточными.

Замечание 2. Признак Вейерштрасса дает абсолютную сходимость, а признаки Дирихле и Абеля обычно используют при доказательстве условной сходимости.

4. Признак Дини.

Пусть функция \(\displaystyle f(x,\gamma )\) непрерывна и неотрицательна в полуполосе \(\displaystyle \left[ a,\mathbf{B} \right)\times \left[ {{\gamma }_{1}},{{\gamma }_{2}} \right]\), и пусть для каждого \(\displaystyle \gamma \in \left[ {{\gamma }_{1}},{{\gamma }_{2}} \right]\) сходится несобственный интеграл \(\displaystyle I(\gamma )=\int\limits_{a}^{\mathbf{B}}{f(x,\gamma )dx}\), причем функция \(\displaystyle I(\gamma )\) непрерывна на \(\displaystyle \left[ {{\gamma }_{1}},{{\gamma }_{2}} \right]\). Тогда интеграл \(\displaystyle I(\gamma )=\int\limits_{a}^{\mathbf{B}}{f(x,\gamma )dx}\) сходится равномерно по \(\displaystyle \gamma \) на \(\displaystyle \left[ {{\gamma }_{1}},{{\gamma }_{2}} \right]\).
